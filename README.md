# sly-browserify
A fork of the excellent [sly.js](https://github.com/darsain/sly) Pegged @1.2.1 to be used with [browserify](http://browserify.org)

# usage
```shell
$ npm install sly-browserify
```
```javascript
// in browser
require('sly-browserify')($, window); // Sly is now global
```
